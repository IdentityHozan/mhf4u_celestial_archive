MODEL II / RECIPROCAL SQUARED

Height from Base of Bus Chassis as a Function of Horizontal Distance from the Leftmost Window Frame

★★★★★
↗ Open in Desmos
"I chose this photo because it shows the Attack on Titan mural on the KL Hop-On Hop-Off bus, which connects to my experience in Malaysia and my interest in anime culture. While observing the Colossal Titan design, I noticed that the back of the cape drawing had a steep curved shape that gradually flattened, similar to the right branch of a reciprocal squared function."
Parent Function\[ y=\frac{1}{x^2} \]
Transformed Model\[ f(x)=\frac{1}{x^2-142}+3.3 \]

Transformations Applied

I selected the parent function \(y=\frac{1}{x^2}\) because its branch has a steep decrease near a vertical asymptote and then gradually levels out. This matched the back of the cape drawing on the bus, which drops quickly near the left side of the selected region before flattening toward the right.

  • Denominator Adjustment: The denominator was changed from \(x^2\) to \(x^2-142\). This creates vertical asymptotes at \(x=\pm\sqrt{142}\), or approximately \(x=\pm11.92\). Since the modeled shoulder segment lies on the right branch, the important asymptote is \(x\approx11.92\).
  • Right-Branch Selection: I restricted the model to values just greater than \(x=11.92\), allowing only the right branch of the reciprocal squared curve to appear. This isolates the part of the function that matches the Titan’s shoulder.
  • Vertical Translation (\(c=3.3\)): The graph was shifted 3.3 units upward so that the curve aligned with the shoulder’s height above the base of the bus chassis.

The final model is \(f(x)=\frac{1}{x^2-142}+3.3\). This equation fits the visible shoulder segment because it captures the steep curved drop near the asymptote and the gradual flattening of the shoulder as it extends to the right.

Domain, Range, and Asymptotic Behaviour

The parent function \(y=\frac{1}{x^2}\) has a domain of \(x\neq0\), a range of \(y>0\), and a vertical asymptote at \(x=0\). In my transformed model, the denominator cannot equal zero, so the full transformed function is undefined at \(x=\pm\sqrt{142}\).

Vertical Asymptotes\[ x=\pm\sqrt{142}\approx\pm11.92 \]
Horizontal Asymptote\[ y=3.3 \]

For the real image, I restricted the domain to only the visible section of the Titan’s shoulder:

\[D:\{x\in\mathbb{R}\mid11.94\le x\le14.2\}\]

This domain begins slightly to the right of the vertical asymptote at \(x\approx11.92\), avoiding undefined values while still capturing the steep start of the shoulder curve. It ends at \(x=14.2\), where the selected shoulder segment stops in the photograph.

Since the function is continuous and decreasing on this restricted interval, the maximum height occurs at \(x=11.94\), and the minimum height occurs at \(x=14.2\).

  • Maximum Height: At \(x=11.94\), \(f(11.94)\approx5.07\).
  • Minimum Height: At \(x=14.2\), \(f(14.2)\approx3.32\).
Restricted Domain\[ 11.94\le x\le14.2 \]
Approximate Range\[ 3.32\le y\le5.07 \]

COSINE

Height of the Building Arch as a Function of Distance

★★★★★
↗ Open in Desmos
"I chose this photo because the smooth curve of the building’s arch was a perfect match for modeling with a trigonometric function. To model the arch appearing in the photo, I began by identifying that its shape closely resembled a cosine wave segment, so I selected the parent function \( y=\cos(x) \) as a starting point. From observing the orientation of the curve in my image, I noticed it peaked at the center rather than the edges, which aligned naturally with the shape of the cosine parent function."
Parent Function\[ y = \cos(x) \]
Transformed Model\[ f(x) = \frac{5}{8}\cos(x - 1.5) + 1 \]

Transformations Applied

The transformation process was a strategic sequence designed to align the theoretical wave with the architectural dimensions of the building:

  • Vertical Compression (\(a = 5/8\)): Initially, determining the vertical scale was challenging because the arch was positioned far from the origin. By temporarily isolating the curve's amplitude in Desmos, I determined a compression factor of \(5/8\) was necessary to match the gentle rise and fall of the physical structure compared to the steeper 1-unit amplitude of the parent function.
  • Horizontal Translation (\(d = 1.5\)): I shifted the graph 1.5 units to the right (\(x - 1.5\)) to align the vertex of the cosine curve with the highest point of the arch. Each unit in this model represents the width of one window pane (approx. 0.8m) to maintain consistent real-world scaling.
  • Vertical Translation (\(c = 1\)): I applied a vertical translation of 1 unit upward (\(+1\)) so the curve’s baseline matched the arch’s height relative to the window base.

Physical Constraints

The domain is strictly restricted to \( D: \{x \in \mathbb{R} \mid 0.25 \le x \le 2.8\} \). I limited the domain to the portion of the arch visible in the photograph, representing the horizontal span measured from the white column base. While the parent cosine function has an infinite domain, the physical arch only exists between these two points; extending the function further would no longer accurately reflect the building's structure.

The parent function \( y=\cos(x) \) has a range of \( R: \{y \in \mathbb{R} \mid -1 \le y \le 1\} \). After applying transformations, the theoretical range of the model becomes \( R: \{y \in \mathbb{R} \mid 0.375 \le y \le 1.625\} \). However, since the domain is restricted to a specific segment of the wave, the practical range is determined by evaluating the function at the domain's boundaries and the vertex.

The range calculation was performed by analyzing the behavior of the function over the interval \([0.25, 2.8]\):

  • Maximum Height: The peak occurs at the vertex \( x=1.5 \), resulting in a height of \( f(1.5)=1.625 \) units.
  • Minimum Height: Since the function is continuous, I evaluated the endpoints at \( x=0.25 \) and \( x=2.8 \), yielding approximate heights of \( 1.197 \) and \( 1.167 \) respectively.
  • Contextual Meaning: These values represent the real-world elevation of the arch measured from the base of the window, where each unit corresponds to a calibrated physical measurement.
Restricted Domain\[ 0.25 \le x \le 2.8 \]
Approximate Range\[ 1.167 \le y \le 1.625 \]

MODEL VI / EXPONENTIAL

Height of Concrete Path Curve as a Function of Distance

★★★★★
↗ Open in Desmos
"I chose this photo because the curve of the concrete path matches the shape of an exponential function. I used the parent function \(y=2^x\) as my starting model because it has the same increasing and upward-curving trend as the path."
Parent Function\[ y = 2^x \]
Transformed Model\[ f(x) = 1.38^{(x-15.6)} - 0.0053 \]

Transformations Applied

After observing the curve’s position and direction in the photograph, I noticed that the path rises gradually from left to right. The standard parent function \(y=2^x\) increased too quickly and did not align with the visible curve, so I adjusted the base, horizontal position, and vertical position of the graph.

  • Base Adjustment (\(b = 1.38\)): The base was changed from 2 to 1.38 to reduce the steepness of the graph. This made the exponential growth more gradual and better matched the curve of the concrete path.
  • Horizontal Translation (\(d = 15.6\)): The graph was shifted 15.6 units to the right so that the curve aligned with the visible starting position of the path in the photograph.
  • Vertical Translation (\(c = -0.0053\)): The graph was shifted 0.0053 units downward so that the lower part of the curve matched the concrete base more accurately.

Through these adjustments, the final model became \(f(x)=1.38^{(x-15.6)}-0.0053\), which closely follows the shape and position of the curve shown in the image.

Domain, Range, and Horizontal Asymptote

The restricted domain is \(D: \{x \in \mathbb{R} \mid 0 \leq x \leq 20\}\). This restriction is used because the visible section of the concrete path in the photograph only extends from \(x=0\) to \(x=20\). The model is only meant to represent this specific part of the real structure.

The parent exponential function has a domain of all real numbers and a range of \(y>0\). After applying the vertical shift, the transformed function has a horizontal asymptote at \(y=-0.0053\). This means the graph approaches this line as \(x\) becomes very small, but it does not touch or cross it. Therefore, the natural range of the transformed function is \(y>-0.0053\).

However, because this model is restricted to the real-world section \(0 \leq x \leq 20\), the actual range is based only on the function values at the endpoints:

  • Starting Height: At \(x=0\), \(f(0) \approx -0.0040\).
  • Maximum Height: At \(x=20\), \(f(20) \approx 4.1201\).
Restricted Domain\[ 0 \le x \le 20 \]
Approximate Range\[ -0.0040 \le y \le 4.1201 \]

Parent Function\[ f(x) = |x| \]

MODEL IX / SQUARE ROOT

Height of the Curved Ceiling Recess as a Function of Horizontal Displacement from Right Foot

★★★★★
↗ Open in Desmos
"I chose this photograph because I took it during the Merdeka celebration, and I wanted this mathematical model to represent Malaysia, its culture, and the architectural details surrounding the event. While analyzing the ceiling structure, I noticed that the curved recess closely resembled the shape of a square root function because of its rapid initial curve followed by a gradual flattening. As a result, I selected the parent function \(y=\sqrt{x}\) as the foundation for my model."
Parent Function \[ y=\sqrt{x} \]
Transformed Model \[ f(x)=0.99\sqrt{-\frac{1}{2}(x-4)}+7 \]

Transformations Applied

The parent square root function normally begins at the origin and increases toward the right. However, the ceiling recess in the image curves toward the left and is positioned significantly higher than the parent graph. To accurately align the mathematical model with the real architectural feature, multiple transformations were applied in a logical sequence.

  • Horizontal Reflection: The negative sign inside the radical reflects the graph across the vertical axis, causing the square root curve to open toward the left instead of the right. This matched the orientation of the ceiling recess visible in the photograph.
  • Horizontal Stretch by a Factor of 2: The coefficient \(-\frac12\) inside the radical horizontally stretches the graph, allowing the curve to widen and more accurately follow the gradual curvature of the recess across the ceiling.
  • Horizontal Translation (\(d=4\)): The expression \((x-4)\) shifts the endpoint of the square root graph 4 units to the right. In the context of this model, the horizontal axis represents displacement measured from my right foot, allowing the endpoint of the curve to align with the architectural boundary of the recess.
  • Vertical Stretch (\(a=0.99\)): The multiplier 0.99 slightly adjusts the height of the graph. Since the value is close to 1, the overall shape remains similar to the parent function while still refining the curve enough to better match the photograph.
  • Vertical Translation (\(c=7\)): The graph was translated 7 units upward so that the entire curve aligned with the elevated ceiling structure in the image.

Through these transformations, the final equation became \(f(x)=0.99\sqrt{-\frac{1}{2}(x-4)}+7\), which closely follows both the direction and curvature of the recessed architectural feature.

Domain, Range, and Restrictions

The parent square root function \(y=\sqrt{x}\) has a domain of \(x\ge0\) and a range of \(y\ge0\). However, because my transformed model contains a reflected and translated radical expression, the restrictions must be re-evaluated.

Radicand Restriction \[ -\frac12(x-4)\ge0 \]
Valid x-values \[ x\le4 \]

Since square root functions cannot contain negative values inside the radical, the graph cannot extend beyond \(x=4\). At \(x=4\), the graph reaches its endpoint because the radicand becomes exactly zero.

The restricted domain for the physical model is:

\[ D:\{x\in\mathbb{R}\mid3.24\le x\le4\} \]

This restriction was chosen because the equation only models the visible portion of the ceiling recess shown in the photograph. Extending the graph further would no longer accurately represent the real architectural structure.

The range was determined by evaluating the function over the restricted interval. Since the graph decreases from left to right across this domain, the maximum value occurs at the left endpoint and the minimum occurs at the square root endpoint at \(x=4\).

  • Maximum Height: At \(x=3.24\), \[ f(3.24)\approx7.6103 \]
  • Minimum Height: At \(x=4\), \[ f(4)=7 \]
Restricted Domain \[ 3.24\le x\le4 \]
Approximate Range \[ 7\le y\le7.6103 \]
