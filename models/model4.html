Transformations Applied

The parent secant function normally has repeating branches separated by vertical asymptotes. To make the curve match the visible arched outlines of the mural, transformations were applied to the parent function in strict mathematical order:

  • Vertical Compression (\(a = 0.99\)): Applied first, the value of 0.99 slightly adjusts the height of the secant branch. Since this value is very close to 1, the graph keeps almost the same shape as the parent function while allowing the curve to align more closely with the top of the mural outline.
  • Horizontal Compression (\(k = \frac{5}{3}\)): The factor of \(\frac{5}{3}\) compresses the graph horizontally. This makes the secant branch narrower and steeper, which helps it accurately follow the sharper upward curves of the background panels.
  • Horizontal Translation (\(d = 1.89\)): The factored expression shifts the graph 1.89 units to the right, moving the secant branch away from the y-axis to align with the mural curve sections being modeled. On this plane, each horizontal grid step represents exactly \(40\text{ cm}\) in real space (calibrated because the \(2.4\text{ m}\) wide background canvas spans \(6\) units), matching the width of a standard desktop computer keyboard.
  • Vertical Translation (\(c = 2.79\)): Finally, the graph is shifted 2.79 units upward. This places the secant branch at the correct vertical height relative to my shoulder-level baseline. Each vertical grid unit represents approximately \(52.4\text{ cm}\), calibrated because my physical height of \(178\text{ cm}\) measures exactly \(3.4\) units in the photograph, which corresponds to the height of a standard carry-on suitcase.

After transformations are executed, the locations of these mathematical discontinuities shift. On our localized workspace branch, the operational asymptotes are located at approximately \( x \approx 0.95 \) and \( x \approx 2.83 \).
