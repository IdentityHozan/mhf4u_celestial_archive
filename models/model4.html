Parent Function\[ y = \sec(x) \]
Transformed Model\[ f(x) = 0.99\sec\left(\frac{5}{3}(x - 1.89)\right) + 2.79 \]

Transformations Applied

The parent secant function normally has repeating branches separated by vertical asymptotes. To make the curve match the visible arched outlines of the mural, transformations were applied to the parent function in strict mathematical order:

  • Vertical Compression (\(a = 0.99\)): Applied first, the value of 0.99 slightly compresses the height of the secant branch vertically. Since this value is very close to 1, the graph keeps almost the same shape as the parent function while allowing the curve to align more closely with the top of the mural outline.
  • Horizontal Compression (\(k = \frac{5}{3}\)): The factor of \(k = \frac{5}{3}\) compresses the graph horizontally by a factor of \(\frac{1}{k} = \frac{3}{5}\). This horizontal compression narrows the secant branch, increasing its steepness to accurately follow the sharper upward curves of the background panels.
  • Horizontal Translation (\(d = 1.89\)): The factored expression shifts the graph 1.89 units to the right (\(d = 1.89\)), moving the central secant branch away from the y-axis to align perfectly with the symmetry axis of the mural curve sections being modeled.
  • Vertical Translation (\(c = 2.79\)): Finally, the graph is shifted 2.79 units upward. This fields a baseline shift that lifts the local minimum of the central branch to the correct vertical height relative to my shoulder-level baseline.

Domain, Range, and Restrictions

The parent function \( y=\sec(x) \) is undefined wherever \( \cos(x)=0 \), creating vertical asymptotes at odd multiples of \(\frac{\pi}{2}\). After transformations are executed, these mathematical discontinuities shift according to the inner argument restriction: \(\frac{5}{3}(x - 1.89) \neq \frac{\pi}{2} + n\pi\). On our localized workspace branch, the operational asymptotes are located at approximately \( x \approx 0.95 \) and \( x \approx 2.83 \).

Vertical Asymptotes\[ x \approx 0.95, \quad x \approx 2.83 \]
Baseline Shift\[ y = 2.79 \]

Since the mural only contains limited visible curved features, the model must be restricted to the specific parts of the graph that actually trace the artwork. The selected domain uses two intervals joined by a union operation because the model follows both the left and right arched sections symmetrically:

\[ D: \{x \in \mathbb{R} \mid -4.36 \le x \le -3.35 \cup 3.3 \le x \le 4.36\} \]

These restrictions prevent the secant graph from continuing infinitely or entering areas where there is no physical mural curve to represent. Within these intervals, the model remains continuous and follows only the visible orange outlines, avoiding the infinite asymptotic spikes that occur as the function approaches its vertical discontinuities.

  • Minimum Curve Height: At the lower outer boundary of the traced section, around \(x=-4.36\), the function evaluates to an approximate vertical displacement of \(f(x)\approx0.98049\).
  • Maximum Curve Height: Near the top visible part of the mural curve, the function reaches an approximate vertical displacement of \(f(x)\approx1.8\).

Therefore, the infinite behaviour of the original parent function is limited to a realistic interval that matches the physical boundaries of the artwork. Instead of representing the entire natural secant graph, this comparison highlights how the real-world model cuts off the mathematical curves precisely where the physical lines of the mural paneling terminate.

Why This Model Fits and Rejected Alternatives

A secant function was selected because the mural curve forms separated arch-like branches with steep outer behaviour. Since \(y=\sec(x)\) naturally creates repeating curved branches divided by vertical asymptotes, it is appropriate for modeling the separated mural shapes. The domain is restricted to two intervals because only those parts of the secant graph correspond to the visible orange mural outlines. A cosine model was considered because the curve has an arch shape, but cosine would produce a continuous wave without the same asymptotic branch behaviour. A parabola was also less suitable because it would only model one simple arch and would not naturally represent the separated left and right sections. The secant model fits better because its reciprocal trigonometric behaviour explains both the curved shape and the need for interval restrictions.
