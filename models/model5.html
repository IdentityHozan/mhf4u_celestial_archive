Parent Function\[ y = \cos(x) \]
Transformed Model\[ f(x) = \frac{5}{8}\cos(x - 1.5) + 1 \]

Transformations Applied

To accurately track the structural curve against the image, the transformations were applied to the parent function in the correct mathematical order:

  • Vertical Compression (\(a = 5/8\)): This was applied first to reduce the amplitude of the wave from 1 to 5/8. This vertical scaling changes the wave's height so that it conforms to the shallow, gentle slope of the concrete archway.
  • Horizontal Compression/Stretch (\(k = 1\)): The horizontal compression factor remains \(\frac{1}{1} = 1\), as the natural horizontal frequency of the parent cosine wave perfectly aligned with the architectural span of the arch without further alteration.
  • Horizontal Translation (\(d = 1.5\)): The graph was shifted 1.5 units to the right (\(d = 1.5\)) to line up the phase peak (vertex) of the cosine wave perfectly with the highest point of the window arch on the grid.
  • Vertical Translation (\(c = 1\)): The graph was shifted 1 unit upward so that its baseline shifts up from the x-axis, matching the real position of the window ledge structure.

Physical Constraints

The domain of this model is restricted to \( D: \{x \in \mathbb{R} \mid 0.25 \le x \le 2.8\} \). This restriction is necessary because the physical arch does not ripple across the entire wall space infinitely like the parent function; it is a localized architectural feature bounded by the column structure on the building facade.

While the parent function naturally fluctuates infinitely between \([-1, 1]\), our transformations change the boundaries of the curve completely. Because this model is restricted over the interval \([0.25, 2.8]\), the actual range is determined directly by evaluating the vertex and its endpoints. Since the absolute maximum vertex is captured within this interval, it defines the upper bound of our range.

  • Maximum Arch Height: The peak occurs exactly at the wave vertex \( x = 1.5 \), where \( f(1.5) = 1.625 \) units.
  • Minimum Arch Height: Since the function is continuous, evaluating the endpoints at \( x = 0.25 \) and \( x = 2.8 \) yields approximate heights of \( 1.1971 \) and \( 1.1672 \) respectively. The absolute lowest boundary occurs at the rightmost edge.
