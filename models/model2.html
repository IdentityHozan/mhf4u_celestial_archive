Parent Function \[ y = \frac{1}{x^2 - 142} \]
Transformed Model \[ f(x) = \frac{1}{x^2 - 142} + 3.3 \]

Transformations Applied

For this model, the accepted parent function is \( y=\frac{1}{x^2-142} \), which is the reciprocal of a quadratic expression. This parent function was selected because its right-hand branch matches the cape curve’s steep rise followed by gradual flattening. Since the parent function already contains the structural shape needed for the cape, the transformation applied to the model is only a vertical translation.

  • Vertical Translation (\(c=3.3\)): The graph was shifted 3.3 units upward, producing \(f(x)=\frac{1}{x^2-142}+3.3\). This raises the curve so that it aligns with the physical height of the cape graphic relative to the lower edge of the bus chassis.

Physical Constraints and Asymptotic Behaviour

The parent function \(y=\frac{1}{x^2-142}\) is undefined wherever its denominator equals zero. Since \(x^2-142=0\) when \(x=\pm\sqrt{142}\), the parent function has two vertical asymptotes: \[ x=\pm\sqrt{142}\approx\pm11.92 \] In this real-world model, only the positive branch is used because the visible cape curve lies to the right of the positive asymptote.

Vertical Asymptotes \[ x=\pm\sqrt{142}\approx\pm11.92 \]
Horizontal Asymptote \[ y=3.3 \]

The full parent function continues infinitely and includes multiple disconnected regions separated by the two vertical asymptotes. However, the physical cape in the photograph only occupies a limited section of the right-hand branch. Therefore, the domain is strictly restricted to the visible cape boundary:

\[ D:\{x\in\mathbb{R}\mid11.94\le x\le14.2\} \]

This restricted domain begins slightly to the right of the positive vertical asymptote at \(x\approx11.92\), preventing the graph from extending into an unrealistic infinite spike. It also ends at \(x=14.2\), where the visible cape curve flattens and terminates in the mural design.

Over the interval \(11.94\le x\le14.2\), the function is continuous and decreasing. Because of this, the maximum and minimum values can be determined by evaluating the function at the domain endpoints.

  • Maximum Cape Height: At the left endpoint near the asymptote, \(x=11.94\): \[ f(11.94)=\frac{1}{11.94^2-142}+3.3\approx5.0743 \] This represents the highest visible part of the cape curve before the graph would rise too sharply toward the asymptote.
  • Minimum Cape Height: At the right endpoint, \(x=14.2\): \[ f(14.2)=\frac{1}{14.2^2-142}+3.3\approx3.3168 \] This represents the lower, flatter part of the cape curve as it moves away from the asymptote.

Therefore, the contextual range of the restricted real-world model is: \[ R:\{y\in\mathbb{R}\mid3.3168\le y\le5.0743\} \] This range is much narrower than the unrestricted rational function because the model only represents the visible cape segment in the photograph.

Why This Model Fits and Rejected Alternatives

A reciprocal of a quadratic function was chosen because the cape curve rises sharply near one boundary and gradually levels out as it moves away from the vertical asymptote. The parent function \(y=\frac{1}{x^2-142}\) naturally has two vertical asymptotes at \(x=\pm\sqrt{142}\), but only the positive branch is used because that is the branch that matches the visible cape. A linear model was rejected because the cape is clearly curved rather than straight. A quadratic model was also less suitable because it would form a rounded turning point instead of the steep asymptotic edge shown in the image. The reciprocal of a quadratic model fits better because it explains both the sharp boundary near the cape’s edge and the gradual flattening as the curve moves rightward.
