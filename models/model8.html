Transformations Applied

For my first choice function slot, I selected the absolute value parent function, \( f(x) = |x| \), to model the sharp V-shaped base of the Caduceus emblem on our banner. To ensure the mathematical model accurately fits the image, the transformations were applied in the correct mathematical order following the mapping structure \(a|k(x-d)|+c\):

  • Vertical Stretch (\(a = 1.3\)): This was applied first because the sides of the graphic form a sharper, narrower angle compared to the standard 45° slope of the parent function, stretching the linear arms vertically.
  • Horizontal Compression/Stretch (\(k = 1\)): The horizontal scaling factor is set at \(\frac{1}{1} = 1\), as the vertical stretch alone was mathematically sufficient to match the slope angle of the graphic vector lines.
  • Horizontal Translation (\(d = 1.05\)): The graph was shifted 1.05 units to the right (\(d = 1.05\)) so that the vertex moves away from the origin and lines up perfectly with the center axis of the banner graphic. This horizontal scale was calculated using the background whiteboard, which is roughly \(130\text{ cm}\) wide and spans \(3.1\) Desmos units. This means each grid unit represents approximately \(42\text{ cm}\), which is roughly equivalent to the width of a standard caution wet floor sign.
  • Vertical Translation (\(c = 2\)): The graph was shifted 2 units upward (\(c = +2\)) to match the physical height of the logo above the floor line baseline. This vertical scale was calibrated using my friend's physical height of \(174\text{ cm}\), which measures \(3.4\) Desmos units on the grid. This sets each vertical unit to roughly \(51\text{ cm}\), which matches the height of a standard carry-on suitcase.

Physical Constraints

The domain of this real-world model is restricted to \( D: \{x \in \mathbb{R} \mid 0.95 \le x \le 1.15\} \). This restriction is necessary because the V-shape does not continue infinitely like the parent function; it only exists as a small part of the overall logo, spanning a physical width on the banner canvas.

Evaluating the function at these specific endpoints gives a restricted range of \( R: \{y \in \mathbb{R} \mid 2 \le y \le 2.13\} \). Comparing this to the parent function's natural range of \( [0, \infty) \) shows how the real-world context changes the boundaries: the lowest vertex point is raised from the ground (\(y=0\)) up to the banner canvas position (\(y=2\)), and the graph is cut off so that it stops tracking where the physical lines of the logo end.

Why This Model Fits and Rejected Alternatives

An absolute value function was selected because the Caduceus base forms a clear V-shape with two straight arms meeting at a vertex. This directly matches the structure of the parent function \(f(x)=|x|\). The vertical stretch makes the arms steeper, while the translations move the vertex to the correct position on the logo. A quadratic function was considered because it can also form a U-like shape, but it would create a rounded bottom rather than the sharp vertex seen in the emblem. A linear function alone was also insufficient because it would only model one side of the V-shape, not both sides together. The absolute value model fits best because it naturally represents two straight segments meeting at one central point.
