Transformations Applied

The parent square root function normally begins at the origin and increases toward the right. However, the ceiling recess in the image curves toward the left and is positioned significantly higher than the parent graph. To accurately align the mathematical model with the real architectural feature, multiple transformations were applied in a logical sequence.

  • Horizontal Reflection: The negative sign inside the radical reflects the graph across the vertical axis, causing the square root curve to open toward the left instead of the right. This matched the orientation of the ceiling recess visible in the photograph.
  • Horizontal Stretch by a Factor of 2: The coefficient \(-\frac12\) inside the radical horizontally stretches the graph, allowing the curve to widen and more accurately follow the gradual curvature of the recess across the ceiling.
  • Horizontal Translation (\(d=4\)): The expression \((x-4)\) shifts the endpoint of the square root graph 4 units to the right. In the context of this model, the horizontal axis represents displacement measured from my right foot, allowing the endpoint of the curve to align with the architectural boundary of the recess.
  • Vertical Stretch (\(a=0.99\)): The multiplier 0.99 slightly adjusts the height of the graph. Since the value is close to 1, the overall shape remains similar to the parent function while still refining the curve enough to better match the photograph.
  • Vertical Translation (\(c=7\)): The graph was translated 7 units upward so that the entire curve aligned with the elevated ceiling structure in the image.

Domain, Range, and Restrictions

The parent square root function \(y=\sqrt{x}\) has a domain of \(x\ge0\) and a range of \(y\ge0\). However, because my transformed model contains a reflected and translated radical expression, the restrictions must be re-evaluated.

Radicand Restriction \[ -\frac12(x-4)\ge0 \]
Valid x-values \[ x\le4 \]

Since square root functions cannot contain negative values inside the radical, the graph cannot extend beyond \(x=4\). At \(x=4\), the graph reaches its endpoint because the radicand becomes exactly zero.

The restricted domain for the physical model is:

\[ D:\{x\in\mathbb{R}\mid3.24\le x\le4\} \]

This restriction was chosen because the equation only models the visible portion of the ceiling recess shown in the photograph. Extending the graph further would no longer accurately represent the real architectural structure.

The range was determined by evaluating the function over the restricted interval. Since the graph decreases from left to right across this domain, the maximum value occurs at the left endpoint and the minimum occurs at the square root endpoint at \(x=4\).

  • Maximum Height: At \(x=3.24\), \[ f(3.24)\approx7.6103 \]
  • Minimum Height: At \(x=4\), \[ f(4)=7 \]
