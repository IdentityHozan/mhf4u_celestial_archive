Transformations Applied

The parent square root function normally begins at the origin and increases toward the right. However, the ceiling recess in the image curves toward the left and is positioned significantly higher than the parent graph. To accurately align the mathematical model with the real architectural feature, transformations were applied in a precise sequence based on the mapping structure \(a\sqrt{k(x-d)}+c\):

  • Horizontal Reflection & Stretch (\(k=-\frac{1}{2}\)): The coefficient \(k=-\frac{1}{2}\) acts inside the radical. The negative sign reflects the graph across the vertical axis, causing the square root curve to open toward the left instead of the right to match the orientation of the ceiling recess. Simultaneously, this parameter applies a horizontal stretch by a factor of \(\frac{1}{|k|} = \frac{1}{1/2} = 2\), widening the path to follow the gradual curvature across the ceiling.
  • Horizontal Translation (\(d=4\)): The factored expression \((x-4)\) outlines a horizontal translation of 4 units to the right (\(d=4\)). In the context of this model, this shifts the starting endpoint of the radical curve away from the origin to align with the architectural boundary line of the recess.
  • Vertical Compression (\(a=0.99\)): The multiplier \(a=0.99\) slightly compresses the height of the graph vertically. Since the value is extremely close to 1, the overall trajectory remains similar to the parent function while refining the edge alignment.
  • Vertical Translation (\(c=7\)): The graph was translated 7 units upward (\(c=+7\)) so that the entire curve aligned with the elevated ceiling structure in the image.

Domain, Range, and Restrictions

The parent square root function \(y=\sqrt{x}\) has a domain of \(x\ge0\) and a range of \(y\ge0\). However, because my transformed model contains a reflected and translated radical expression, the restrictions must be re-evaluated using the foundational radicand requirement:

Radicand Restriction \[ -\frac12(x-4)\ge0 \]
Valid x-values \[ x\le4 \]

Since square root functions cannot contain negative values inside the radical over the real number set, the graph cannot extend beyond its endpoint at \(x=4\), where the interior value evaluates to exactly zero.

The restricted domain for the physical model is:

\[ D:\{x\in\mathbb{R}\mid3.24\le x\le4\} \]

This restriction was chosen because the equation only models the visible portion of the ceiling recess shown in the photograph. Extending the graph further would no longer accurately represent the real architectural structure.

The range was determined by evaluating the function over the restricted interval. Since the graph decreases from left to right across this domain, the maximum value occurs at the left endpoint and the minimum occurs at the square root endpoint at \(x=4\).

  • Maximum Height: At \(x=3.24\), \[ f(3.24) = 0.99\sqrt{-\frac{1}{2}(3.24-4)}+7 \approx 7.6103 \]
  • Minimum Height: At \(x=4\), \[ f(4) = 0.99\sqrt{0}+7 = 7 \]

Why This Model Fits and Rejected Alternatives

A square root function was selected because the ceiling recess has a rapid initial curve that gradually flattens out. This matches the behaviour of \(y=\sqrt{x}\), where the graph changes quickly near its endpoint and then increases more slowly. The negative value inside the radical reflects the graph horizontally so that it opens toward the left, matching the direction of the ceiling curve in the image. A logarithmic function was considered because it also rises quickly and flattens, but the ceiling recess has a clear endpoint, which is better represented by the square root function’s radicand restriction. A quadratic model was less suitable because it would create a symmetric curve rather than the one-sided flattening shape shown in the image. Therefore, the square root model best captures the direction, endpoint, and curvature of the ceiling recess.
