Parent Function \[ y=\log(x) \]
Transformed Model \[ f(x)=\frac{7}{20}\log(41(x+0.0122))+3.98 \]

Transformations Applied

The parent logarithmic function \(y=\log(x)\) starts near the y-axis and increases slowly as x becomes larger. To make the curve match the wall painting, I adjusted the steepness, horizontal placement, and vertical height of the graph.

  • Vertical Compression (\(a=\frac{7}{20}\)): The value \(a=\frac{7}{20}\), or \(0.35\), compresses the graph vertically. This makes the curve less steep so it better matches the gradual rise of the painted wall curve.
  • Horizontal Compression (\(k=41\)): The value \(k=41\) compresses the logarithmic graph horizontally. This makes the curve rise quickly over a short horizontal distance, which matches the sharper starting section of the wall painting.
  • Horizontal Translation (\(d=-0.0122\)): Since the logarithm contains \((x+0.0122)\), the graph shifts \(0.0122\) units to the left. In the form \(a\log(k(x-d))+c\), this means \(d=-0.0122\), because \(x+0.0122=x-(-0.0122)\). On this grid, the x-axis represents horizontal displacement from the left foot base, so this small shift helps align the beginning of the curve with the painted edge.
  • Vertical Translation (\(c=3.98\)): The \(+3.98\) shifts the entire graph upward. This places the logarithmic curve at the correct height on the wall painting instead of keeping it near the x-axis. On the vertical axis, each graph unit represents approximately \(39.5\text{ cm}\), rounded to about \(40\text{ cm}\), because my height of \(162\text{ cm}\) spans about \(4.1\) vertical units in the photograph. This is close to the width of one small guitar, so the y-axis scale helps connect the graph units to a real-world measurement.

Domain, Range, and Asymptotic Behaviour

The parent function \(y=\log(x)\) is only defined when the input of the logarithm is greater than zero. For this model, the input is:

\[ 41(x+0.0122)>0 \]

Since \(41\) is positive, the restriction becomes:

\[ x+0.0122>0 \]

Therefore, the function is only defined when:

\[ x>-0.0122 \]

This means the vertical asymptote is located at \(x=-0.0122\). The graph cannot touch or pass this line because the logarithm would become undefined.

Vertical Asymptote \[ x=-0.0122 \]
Vertical Shift \[ y=3.98 \]

The domain is restricted to the part of the graph that actually traces the visible wall painting curve:

\[ D:\{x\in\mathbb{R}\mid -0.005\le x\le0.5\} \]

This restriction keeps the model on the visible section of the wall painting. It also prevents the graph from moving too close to the vertical asymptote or extending into areas where there is no painted curve to model.

Since the logarithmic function increases over this restricted interval, the minimum and maximum values occur at the endpoints.

  • Minimum Curve Height: At \(x=-0.005\), the function gives \(f(-0.005)\approx3.7944\).
  • Maximum Curve Height: At \(x=0.5\), the function gives \(f(0.5)\approx4.4428\).

Therefore, the approximate range of the model is:

\[ R:\{y\in\mathbb{R}\mid 3.7944\le y\le4.4428\} \]

Contextual Domain \[ -0.005\le x\le0.5 \]
Contextual Range \[ 3.7944\le y\le4.4428 \]
